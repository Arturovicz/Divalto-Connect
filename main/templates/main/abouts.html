Divalto Connect Project

The goal of the project is to obtain easily information from Divalto database into a new datastore. In this new datastore, a new database structure that allows to access to the quantity of stock of each part for each day is defined. In Divalto Original, it’s difficult to know the level of stock for a special day. This particular solution, called Divalto Connect gives to the user 2 key abilities:

  1. Find the desired article, either by name, unique id, department, fournisseur or even per dimensions.
  2. Plot the graph with starting and ending date, searching with article unique ID
The application is built with this 3 components:
  1. An ETL to transform data from the Original Divalto Database: Talend Open Studio.
  2. A Datastore with MySQL Database Engine to store the data extracted
  3. An user-centered interface allowing to access to the information: Python/Django

The retrieved data is the totality of data existing in company ‘998 – Papyrus (Demo)’. The application is secure, using only the Login and Password distributed by ECAM Strasbourg-Europe.
